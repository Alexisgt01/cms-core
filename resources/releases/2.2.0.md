## 🎉 Version 2.2.0 — Mode d'affichage des icônes

Une mise à jour ciblée qui apporte plus de flexibilité dans l'affichage des icônes et corrige un souci technique sur les images. 🎨

### ✨ Nouveautés

- **Mode de rendu des icônes configurable** — Choisissez comment les icônes sont affichées sur votre site : en SVG (image vectorielle intégrée dans la page) ou en classe CSS (chargement via une feuille de style). Cette option permet d'adapter le comportement au thème de votre site.

### 🐛 Corrections

- Correction d'un problème d'affichage sur certaines images dont l'URL contenait des caractères spéciaux (accents, espaces…). Les images s'affichent désormais correctement dans tous les cas.

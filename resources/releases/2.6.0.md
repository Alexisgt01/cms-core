## 🎉 Version 2.6.0 — Gestion des fonctionnalités

Simplifiez votre interface en activant uniquement les modules dont vous avez besoin !

### ✨ Nouveautés

- **Toggles de fonctionnalités** — Un nouvel onglet « Fonctionnalités » dans les Paramètres du site vous permet d'activer ou désactiver chaque module de la barre latérale (Blog, Pages, Médias, Contact, Collections, SEO, etc.).
- **Contrôle par groupe et par élément** — Désactivez un groupe entier (ex : Contact) ou uniquement certains éléments au sein d'un groupe (ex : masquer les Webhooks tout en gardant Contacts et Demandes).
- **Interface réactive** — Les sous-éléments disparaissent automatiquement quand leur groupe parent est désactivé.

### 🔧 Détails techniques

- Les réglages sont stockés en base de données et pris en compte immédiatement après sauvegarde.
- Un fichier de configuration (`cms-features.php`) définit les valeurs par défaut (tout activé).

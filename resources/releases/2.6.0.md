## 🎉 Version 2.6.0 — Personnalisation de la navigation

Simplifiez votre interface en activant uniquement les modules dont vous avez besoin !

### ✨ Nouveautés

- **Personnaliser la navigation** — Un lien en bas de la barre latérale ouvre un panneau permettant d'activer ou desactiver chaque module (Blog, Pages, Medias, Contact, Collections, SEO, etc.).
- **Controle par groupe et par element** — Desactivez un groupe entier (ex : Contact) ou uniquement certains elements optionnels au sein d'un groupe (ex : masquer les Webhooks tout en gardant Contacts et Demandes).
- **Elements obligatoires** — Les elements essentiels d'un module (ex : Contacts, Demandes et Parametres pour le module Contact) restent toujours visibles tant que le module est actif.
- **Interface reactive** — Les sous-elements disparaissent automatiquement quand leur groupe parent est desactive. Les modifications prennent effet immediatement.

### 🔧 Details techniques

- Les reglages sont stockes en base de donnees et pris en compte immediatement apres sauvegarde.
- Un fichier de configuration (`cms-features.php`) definit les valeurs par defaut (tout active).
- Accessible uniquement aux utilisateurs ayant la permission « manage site settings ».

## 🎉 Version 1.4.0 — Import d'images par URL et champ URL intelligent

Gagner du temps, c'est le mot d'ordre de cette version ! Importez des images directement depuis une URL, et profitez d'un tout nouveau champ de lien intelligent qui vous propose vos pages existantes. 🔗

### ✨ Nouveautés

- **Import d'images par URL** — Plus besoin de télécharger une image sur votre ordinateur pour l'ajouter à la médiathèque : collez simplement l'adresse web de l'image et elle est importée automatiquement.
- **Nouveau champ URL pour les sections** — Un type de champ dédié pour ajouter des liens dans vos sections de page.
- **Champ URL intelligent** — Le sélecteur de lien vous propose désormais vos pages existantes et vos routes dans un menu groupé, en plus de la saisie libre pour les liens externes. Fini les erreurs de copier-coller !
- **Email de test** — Un bouton sur le tableau de bord d'administration permet d'envoyer un email de test pour vérifier que la configuration mail de votre site fonctionne correctement.

### 🔧 Améliorations

- Les sections de page sont désormais repliées par défaut pour une meilleure lisibilité lorsque vous avez beaucoup de blocs.
- Meilleure prise en charge des champs URL dans les blocs repliés.

### 🐛 Corrections

- Correction de l'affichage de la grille d'icônes avec les icônes Font Awesome.

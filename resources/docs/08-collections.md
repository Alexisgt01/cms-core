## 📦 Collections

Les collections sont des **contenus structurés personnalisés** créés sur mesure pour votre site. Contrairement aux articles de blog ou aux pages, les collections sont des types de contenu définis spécifiquement pour vos besoins par votre développeur.

---

### 💡 Qu'est-ce qu'une collection ?

Une collection est un ensemble d'**entrées** (éléments) qui partagent la même structure. Pensez-y comme un répertoire spécialisé.

Voici des exemples courants de collections :

| Collection | Contenu de chaque entrée |
|------------|-------------------------|
| **Témoignages** | Nom du client, photo, avis, note, entreprise |
| **Équipe** | Nom, photo, poste, biographie, réseaux sociaux |
| **FAQ** | Question, réponse, catégorie |
| **Partenaires** | Logo, nom, lien vers le site, description |
| **Réalisations** | Titre, images, description, client, date |
| **Services** | Titre, icône, description, prix |

Chaque type de collection a sa propre **entrée dans le menu** latéral, ce qui vous permet d'y accéder directement.

> 💡 **Astuce** : Les types de collections disponibles dépendent de ce que votre développeur a configuré pour votre site. Si vous avez besoin d'un nouveau type de collection, demandez-lui !

---

### 📋 Gérer les entrées d'une collection

#### Voir les entrées

Cliquez sur le nom de la collection dans le menu latéral pour voir la liste de toutes ses entrées. La liste affiche les colonnes pertinentes définies pour ce type de collection.

#### Créer une entrée

1. Cliquez sur **Créer** en haut à droite
2. Remplissez les champs du formulaire — ils sont spécifiques à chaque type de collection
3. Cliquez sur **Créer** pour enregistrer

#### Modifier une entrée

1. Cliquez sur l'entrée dans la liste
2. Modifiez les champs souhaités
3. Cliquez sur **Enregistrer**

#### Supprimer une entrée

1. Cliquez sur l'entrée dans la liste
2. Cliquez sur **Supprimer** dans la barre d'actions
3. Confirmez la suppression

> ⚠️ **Attention** : La suppression d'une entrée est définitive. Assurez-vous de ne plus en avoir besoin avant de la supprimer.

---

### 🔗 Slug (optionnel)

Certaines collections disposent d'un champ **slug** qui permet de créer une URL unique pour chaque entrée. C'est le cas par exemple pour les réalisations ou les services qui ont leur propre page sur le site.

Le slug est généré automatiquement à partir du titre, mais vous pouvez le modifier manuellement.

> 💡 **Astuce** : Si votre collection a un champ slug, cela signifie que chaque entrée a sa propre page accessible sur votre site. Pensez à remplir les champs SEO pour un bon référencement.

---

### 📊 États (optionnel)

Certaines collections supportent les **états de publication**, comme les articles de blog :

| État | Description |
|------|-------------|
| **Brouillon** 📝 | L'entrée est en cours de création. Elle n'est pas visible sur le site. |
| **Publiée** ✅ | L'entrée est en ligne et visible par les visiteurs. |

Pour publier une entrée, utilisez le bouton **Publier** dans la barre d'actions.

> 💡 **Astuce** : Si vous ne voyez pas de bouton « Publier », c'est que cette collection n'utilise pas de système d'états. Les entrées sont alors visibles dès leur création.

---

### ↕️ Trier les entrées

Certaines collections permettent le **tri par glisser-déposer** pour définir l'ordre d'affichage des entrées sur votre site :

1. Dans la liste des entrées, repérez l'icône de **poignée** (≡) à gauche de chaque ligne
2. **Cliquez et maintenez** sur la poignée
3. **Faites glisser** l'entrée vers sa nouvelle position
4. **Relâchez** — l'ordre est enregistré automatiquement

> 💡 **Astuce** : Le tri par position est idéal pour les collections comme « Équipe » (ordre hiérarchique), « Partenaires » (ordre de priorité) ou « Services » (ordre logique de présentation).

---

### 🔍 SEO sur les collections (optionnel)

Si une collection dispose de pages individuelles (avec slug), vous trouverez les mêmes onglets SEO que pour les articles et les pages :

- **SEO** : titre, description, H1, mots-clés
- **Open Graph** : paramètres de partage Facebook/réseaux sociaux
- **Twitter** : paramètres de partage X/Twitter
- **Schema** : données structurées

Consultez la section **SEO** de cette documentation pour plus de détails sur chaque champ.

---

### 🎯 Bonnes pratiques pour les collections

- **Remplissez tous les champs** de chaque entrée pour un affichage optimal sur le site
- **Utilisez des images de qualité** et au bon format pour les photos (équipe, partenaires, réalisations)
- **Ordonnez vos entrées** de manière logique (par importance, par ordre alphabétique, par date)
- **Mettez à jour régulièrement** vos collections (nouveaux témoignages, nouveaux membres d'équipe, etc.)
- **Remplissez le SEO** si la collection dispose de pages individuelles
- **Demandez à votre développeur** si vous avez besoin d'un nouveau type de collection

## 🖼️ Médias

La bibliothèque de médias est l'espace centralisé où vous stockez et gérez tous vos fichiers : images, documents PDF, vidéos, etc. Tous les contenus de votre site (articles, pages, collections) utilisent les médias de cette bibliothèque.

Accédez-y via le menu **Médias → Bibliothèque de médias**.

---

### 🗂️ Naviguer dans la bibliothèque

La bibliothèque vous offre plusieurs façons de retrouver vos fichiers :

#### Dossiers

Organisez vos médias dans des **dossiers** pour les retrouver facilement :

- Cliquez sur **Nouveau dossier** pour en créer un
- Cliquez sur un dossier pour y accéder
- Utilisez le **fil d'Ariane** en haut pour naviguer dans l'arborescence

Exemple d'organisation :

```
📁 Images
   ├── 📁 Blog
   ├── 📁 Équipe
   └── 📁 Produits
📁 Documents
   ├── 📁 Brochures
   └── 📁 Contrats
```

#### Recherche

Utilisez la **barre de recherche** pour trouver un fichier par son nom ou ses tags.

#### Filtres

Affinez votre recherche avec les filtres disponibles :

- **Par type** : images, documents, vidéos, etc.
- **Par taille** : petit, moyen, grand
- **Par date** : date d'ajout

> 💡 **Astuce** : Donnez des noms explicites à vos fichiers avant de les importer. « photo-equipe-2026.jpg » est beaucoup plus facile à retrouver que « IMG_4523.jpg ».

---

### ⬆️ Importer des fichiers

Plusieurs méthodes s'offrent à vous pour ajouter des fichiers à votre bibliothèque :

#### Glisser-déposer

1. Ouvrez la bibliothèque de médias
2. **Faites glisser** vos fichiers depuis votre ordinateur directement dans la zone de la bibliothèque
3. Les fichiers sont importés automatiquement

#### Bouton d'import

1. Cliquez sur le bouton **Importer**
2. Sélectionnez un ou plusieurs fichiers depuis votre ordinateur
3. Validez l'import

#### Import depuis une URL

Pour importer un fichier disponible en ligne :

1. Cliquez sur **Importer depuis une URL**
2. Collez l'adresse du fichier
3. Le fichier sera téléchargé et ajouté à votre bibliothèque

> 💡 **Astuce** : L'import par URL est pratique pour récupérer des images depuis d'autres sites ou des liens partagés par des collaborateurs.

---

### 📸 Intégration Unsplash

Votre CMS dispose d'une intégration avec **Unsplash**, une banque d'images gratuites et libres de droits de haute qualité.

Pour utiliser Unsplash :

1. Cliquez sur le bouton **Unsplash** dans la bibliothèque
2. **Recherchez** une image par mots-clés (en anglais de préférence pour plus de résultats)
3. Parcourez les résultats
4. Cliquez sur l'image qui vous plaît pour l'ajouter à votre bibliothèque

> 💡 **Astuce** : Les images Unsplash sont gratuites et libres de droits pour un usage commercial. C'est une excellente source pour trouver des photos professionnelles pour vos articles et pages.

> ⚠️ **Bon à savoir** : Même si l'attribution n'est pas obligatoire, il est apprécié de créditer le photographe quand c'est possible.

### ✏️ Gérer vos fichiers

#### Renommer un fichier

1. Cliquez sur un fichier pour ouvrir sa fiche
2. Modifiez le champ **Nom**
3. Enregistrez

#### Modifier le texte alternatif (alt)

Le texte alternatif est **essentiel pour le SEO et l'accessibilité**. Il décrit le contenu de l'image pour les moteurs de recherche et les lecteurs d'écran.

1. Cliquez sur le fichier
2. Remplissez le champ **Texte alternatif**
3. Enregistrez

> 💡 **Astuce** : Un bon texte alternatif est descriptif et concis. Par exemple, pour une photo d'équipe : « L'équipe de 8 personnes réunie dans le bureau parisien ». Évitez les formulations comme « Image de... » ou « Photo de... ».

#### Déplacer un fichier dans un dossier

1. Cliquez sur le fichier
2. Sélectionnez le **dossier de destination**
3. Enregistrez

#### Supprimer un fichier

1. Cliquez sur le fichier
2. Cliquez sur **Supprimer**
3. Confirmez la suppression

> ⚠️ **Attention** : La suppression d'un média est définitive. Si ce média est utilisé dans un article ou une page, il ne sera plus affiché.

---

### ✅ Opérations groupées

Pour gagner du temps, vous pouvez effectuer des actions sur **plusieurs fichiers à la fois** :

1. **Sélectionnez** les fichiers souhaités en cliquant sur leur case à cocher
2. Utilisez la **barre d'actions groupées** qui apparaît :
   - 📁 **Déplacer** : déplacer les fichiers sélectionnés vers un dossier
   - 🗑️ **Supprimer** : supprimer les fichiers sélectionnés
   - 🏷️ **Ajouter des tags** : taguer les fichiers pour mieux les organiser

> 💡 **Astuce** : Les opérations groupées sont très pratiques après un gros import de fichiers. Sélectionnez-les tous, puis déplacez-les dans le bon dossier et ajoutez les tags appropriés en une seule action.

---

### 🔗 Utiliser les médias dans vos contenus

Lorsque vous éditez un article, une page ou tout autre contenu, vous verrez un composant **Sélecteur de média** (MediaPicker) qui vous permet de :

1. **Parcourir** votre bibliothèque de médias
2. **Rechercher** un fichier par son nom
3. **Sélectionner** le fichier souhaité
4. **Importer** un nouveau fichier directement depuis le formulaire si besoin

Le sélecteur de média affiche un **aperçu** du fichier sélectionné directement dans le formulaire.

---

### 🎯 Bonnes pratiques pour les médias

- **Organisez vos fichiers** dans des dossiers thématiques dès le départ
- **Nommez vos fichiers** de manière descriptive avant l'import
- **Remplissez systématiquement le texte alternatif** de vos images (SEO + accessibilité)
- **Optimisez la taille** de vos images avant de les importer (les images trop lourdes ralentissent le site)
- **Utilisez Unsplash** pour trouver des photos de qualité professionnelle gratuitement
- **Supprimez les fichiers inutilisés** régulièrement pour garder votre bibliothèque propre
- **Utilisez les tags** pour retrouver facilement vos fichiers (ex : « bannière », « portrait », « produit »)

## 🚀 Premiers pas

Bienvenue dans votre espace d'administration ! Ce guide va vous aider à prendre en main votre CMS en quelques minutes. Pas besoin de compétences techniques : tout se fait depuis une interface visuelle simple et intuitive.

---

### 🧭 Naviguer dans le panneau d'administration

Votre panneau d'administration est organisé autour de deux zones principales :

- **La barre latérale gauche** : c'est votre menu principal. Vous y trouverez toutes les sections de votre site (Blog, Pages, Médias, etc.). Cliquez sur un élément pour accéder à sa gestion.
- **Le menu utilisateur (en haut à droite)** : en cliquant sur votre nom ou votre avatar, vous accédez à votre profil, vos préférences et au bouton de déconnexion.

> 💡 **Astuce** : Sur mobile ou petit écran, la barre latérale se replie automatiquement. Cliquez sur l'icône ☰ pour l'ouvrir.

---

### 📂 Les grandes sections de votre CMS

Voici un aperçu de ce que vous trouverez dans chaque section :

| Section | Description |
|---------|-------------|
| **📊 Tableaux de bord** | Vue d'ensemble de l'activité de votre site (statistiques, derniers contenus, etc.) |
| **📝 Blog** | Gérez vos articles, auteurs, catégories et tags |
| **📄 Pages** | Créez et organisez les pages de votre site avec un constructeur de sections |
| **🖼️ Médias** | Votre bibliothèque de fichiers (images, documents, vidéos) |
| **🔍 SEO** | Optimisez votre référencement naturel et gérez vos redirections |
| **📦 Collections** | Gérez des contenus structurés personnalisés (témoignages, FAQ, équipe, etc.) |
| **✉️ Contact** | Consultez les messages reçus via vos formulaires de contact |
| **⚙️ Administration** | Gérez les utilisateurs, les rôles, les réglages du site et le journal d'activité |

---

### 📊 Le tableau de bord

Lorsque vous vous connectez, vous arrivez sur votre **tableau de bord**. Il vous donne une vue rapide de l'état de votre site :

- **Tableau de bord Blog** : nombre d'articles publiés, programmés, en brouillon, graphique mensuel des publications et liste des derniers articles.
- **Tableau de bord Admin** : nombre d'utilisateurs, de rôles, de médias et de dossiers, ainsi que la liste des derniers utilisateurs inscrits.

> 💡 **Astuce** : Consultez votre tableau de bord régulièrement pour garder un œil sur l'activité de votre site.

---

### 👤 Modifier votre profil

Pour mettre à jour vos informations personnelles :

1. Cliquez sur **votre nom** en haut à droite de l'écran
2. Sélectionnez **Profil** dans le menu déroulant
3. Modifiez les informations souhaitées :
   - Votre **nom**
   - Votre **adresse e-mail**
   - Votre **mot de passe** (laissez vide pour ne pas le changer)
   - Votre **avatar**
4. Cliquez sur **Enregistrer**

> 💡 **Astuce** : Choisissez un mot de passe robuste (au moins 8 caractères, avec des majuscules, chiffres et caractères spéciaux) pour protéger votre compte.

---

### 🔑 Se déconnecter

Pour vous déconnecter en toute sécurité :

1. Cliquez sur **votre nom** en haut à droite
2. Cliquez sur **Se déconnecter**

> ⚠️ **Important** : Pensez toujours à vous déconnecter lorsque vous utilisez un ordinateur partagé.

---

### 🎯 Par où commencer ?

Voici un parcours suggéré pour bien démarrer :

1. **Modifiez votre profil** pour personnaliser votre compte
2. **Explorez les réglages du site** (section Administration → Réglages du site)
3. **Ajoutez vos premiers médias** dans la bibliothèque
4. **Créez votre première page** ou votre premier article de blog
5. **Vérifiez le SEO** de vos contenus avant de les publier

Bonne prise en main ! 🎉
